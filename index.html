Clinical Evidence Management System

This simulation trains APA citation as part of an **evidence management workflow** used in healthcare communication. Evidence lifecycle: Triage → Source Intake → Documentation → Citation Signals → Integration → Evidence Stewardship Module 1 focuses on the first step: **recognizing when a claim requires evidence documentation.**
